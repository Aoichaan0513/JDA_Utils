# JDA_Utils

JDA (Java Discord API) Utilities library.

## Download

### Gradle DSL

```gradle
repositories {
    maven { url 'https://repository.aoichaan0513.jp' }
}

dependencies {
    // implementation 'jp.aoichaan0513:JDA_Utils:1.0.4'
    implementation group: 'jp.aoichaan0513', name: 'JDA_Utils', version: '1.0.4'
}
```

### Kotlin DSL

```gradle
repositories {
    maven("https://repository.aoichaan0513.jp")
}

dependencies {
    // implementation("jp.aoichaan0513:JDA_Utils:1.0.4")
    implementation("jp.aoichaan0513", "JDA_Utils", "1.0.4")
}
```
